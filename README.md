## Installation

### Client Side

1. Fork the repo

2. Run the commands on cmd line/terminal
    ```
    npm install
    ```

3. To run the server
    ```
    npm run dev
    ```

### Server Side
1. Run the commands on cmd line/terminal
    ```
    npm install
    ```

2. To run the server
    ```
    npm start
    ```
